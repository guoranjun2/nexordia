# Task: Add attribute name and value search tools
- **Scope:** Add tools to list attribute names for a map and to search nodes by attribute name and value.
- **Motivation:** Attribute search needs a map specific attribute name list so the model can search reliably.
- **Research summary:**
  - Review how attribute names are stored on map nodes and whether there is a central registry.
  - Review how attribute values are compared for search and how case sensitivity or matching modes apply.
- **Design:**
  - Add a tool that returns all attribute names available in a map.
  - Add a tool that searches nodes by attribute name and value, returning matching node identifiers.
  - Return identifiers and short texts for matching nodes to reduce follow up reads.
- **Test specification:**
  - Verify attribute name listing returns all unique names for a map.
  - Verify attribute search finds matching nodes by name and value.
  - Verify responses include identifiers and short texts for matching nodes.
