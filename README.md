# Nexordia

Nexordia is a free and open source mind mapping and knowledge management application. It helps structure ideas, notes, tasks, references, and project knowledge in connected visual maps.

Nexordia is a fork of [Freeplane](https://github.com/freeplane/freeplane), based on the Freeplane 1.13.x code line. The project keeps the original Freeplane copyright and license notices, and adds independent development, packaging, and release management under the Nexordia name.

## Status

Current Nexordia version:

```text
0.1.0
```

This is an early independent release line. Some internal package names, plugin identifiers, file format names, and compatibility strings still use the original Freeplane technical names while the external product identity moves to Nexordia.
